# Custom WordPress Theme 

## Description
A simple, custom WordPress theme developed without the use of any plugins. 

### Features
- Theme logo
- Search bar  
- Custom Widgets
- The Loop
- Featured Images
- Blog Comments

### Page Templates
- index.php
- page.php
- single.php
- category.php
- tag.php
- author.php
- archive.php
- search.php
- 404.php

### Custom Sections 
- header.php
- footer.php
- sidebar.php
- comments.php

### Technologies Used
- PHP 
- HTML 
- CSS

### Integrations
- WordPress API
